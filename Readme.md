### NGINX

Trouver le fichier de conf : nginx -t

Linux:
lancer nginx
`service nginx status`

Arreter nginx
`service nginx status`

Mac (brew):
lancer nginx
`brew services start nginx`
ou
`sudo nginx`

Arreter nginx
`brew services stop nginx`
ou
`sudo nginx -s stop`
